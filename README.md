## How to Run

1. Clone this repository to your local machine.
   ```bash
   git clone https://github.com/TranKienCuong2003/30-Days-of-JavaScript
   ```
2. Navigate to the project directory
   ```bash
   cd 30-Days-of-JavaScript
   ```
3. Open the solutions in your preferred editor (e.g., Visual Studio Code) and run the JavaScript files using Node.js or directly in the browser console.
4. You can also run each solution individually by executing:
   ```bash
   node filename.js
   ```
